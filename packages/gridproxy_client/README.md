# gridproxy_client

This a Typescript client to communicate with GridProxy based on the network.

## Installation

```bash
yarn install
```

## Building

```bash
yarn build
```

### Supported URLs

- [devnet](wss://tfchain.dev.grid.tf)
- [qanet](wss://tfchain.qa.grid.tf)
- [testnet](wss://tfchain.test.grid.tf)
- [mainnet](wss://tfchain.grid.tf)
