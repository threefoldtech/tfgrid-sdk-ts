# UI

Threefold UI is a project that contains reusable web components created by Threefold organization.

## Related Docs

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```
